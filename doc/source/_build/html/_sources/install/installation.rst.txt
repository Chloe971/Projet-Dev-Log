Installation
=============

Run a terminal and then : 

.. role:: bash(code)
   :language: bash

   
Install from Pypi
^^^^^^^^^^^^^^^^^^^
.. code-block:: bash

    pip install -i https://test.pypi.org/simple/ draw-circle-fourier


Dependencies
^^^^^^^^^^^^^
In order to fully use this package, you have to install these following libraries : 

* FFMPEG: run :bash:`pip install ffmpeg`
