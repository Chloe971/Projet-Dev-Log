class PACKAGE.draw_circle_fourier.ImageAnimation.AnimationImage.DrawAnimation(x_DFT, y_DFT, coef, order, space, fig_lim)¶

Creates an animation that draws a curve representing an image using Fourier coefficients.

Parameters

  • x_DFT (list.) – Real part of the Fourier transform.

  • y_DFT (list.) – Imaginary part of the Fourier transform.

  • coef (ndarray.) – Fourier coefficients.

  • order (int.) – Variable that we use to determine the number of Fourier coefficients that we will generate.

  • space (tuple.) – Variable used to give the number of images for the animation

  • fig_lim (list.) – Define the limit of the x and y axes of our plot.
