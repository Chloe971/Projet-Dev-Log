Coefficients of Fourier¶

In this part, we will find the Fourier coefficients to locate approximately given points.

To do this, we have created a function class named FOURIER for our Package draw_circle_fourier

Important note on numbers order selection:¶

When there are more than 500 coefficients ask, the program takes time and the image is no longer clear.

The class methods¶

class PACKAGE.draw_circle_fourier.CoeffFourier.CoeffFourier.Fourier(time_table, x_table, y_table, order)¶
DFT(t, coef_list, order)¶

:param t : time

:param coef_list : complex Fourier coefficients in the form of an (array) of elementary values

:param order : Variable that we use to determine the number of Fourier coefficients that we will generate.

rtype : It will return as output parameters, the real part and the imaginary part of the transform.

coef_list(time_table, x_table, y_table, order)¶

Convert coordinates found by ImageReader class to complex numbers with real and imaginary part.

Parameters

  • time_table (tuple) – Time list from 0 to 2PI.

  • x_table (float) – Coordinate of the X axis.

  • y_table (float) – Coordinate on the Y axis

  • order (int) – Variable that we use to determine the number of Fourier coefficients that we will generate.

:rtype : Desired complex Fourier coefficients in the form of an (array) of elementary values
